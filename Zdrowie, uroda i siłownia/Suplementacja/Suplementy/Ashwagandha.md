## W skrócie

[[Ashwagandha]] to roślina adaptogenna używana głównie pod stres, napięcie i sen. Może mieć sens u niektórych osób, ale nie jest neutralna. To nie jest „naturalna tabletka bez ryzyka”.

Najważniejsza zasada: **testować ostrożnie i obserwować nastrój, sen, energię oraz objawy niepożądane**.

## Co to ma robić

- zmniejszać odczuwany stres,
- wspierać wyciszenie,
- u części osób poprawiać sen,
- potencjalnie wpływać na kortyzol i reakcję stresową,
- działać subtelnie, nie jak lek uspokajający.

## Kiedy może mieć sens

- okres wysokiego napięcia,
- trudność z wyciszeniem wieczorem,
- stres bez konkretnej ostrej przyczyny medycznej,
- świadomy test przez ograniczony czas,
- gdy nie pogarsza napędu, nastroju ani snu.

## Kiedy uważać albo odpuścić

- choroby wątroby,
- choroby tarczycy lub leki na tarczycę,
- choroby autoimmunologiczne,
- leki uspokajające, nasenne lub przeciwlękowe,
- ciąża,
- dziwne pogorszenie nastroju, apatia albo spłaszczenie emocji,
- objawy ze strony wątroby: zażółcenie skóry, ciemny mocz, jasny stolec, ból pod prawym łukiem żebrowym, silne zmęczenie.

## Możliwe efekty uboczne

| Objaw | Co zrobić |
|---|---|
| senność | sprawdzić porę i dawkę |
| mdłości / biegunka | przerwać lub zmniejszyć dawkę |
| ból głowy | obserwować, nie zwiększać dawki |
| apatia / spłaszczenie | przerwać test |
| objawy wątrobowe | przerwać i skonsultować medycznie |

## Jak testować

| Krok | Co zrobić |
|---|---|
| 1 | nie dokładać równolegle innych adaptogenów |
| 2 | zapisać start w [[Dziennik suplementacji]] |
| 3 | obserwować sen, napięcie, nastrój i energię |
| 4 | ocenić po 2–4 tygodniach |
| 5 | jeśli pogarsza napęd albo nastrój — odstawić |

## Czego nie robić

- nie brać bezterminowo tylko dlatego, że jest „naturalna”,
- nie używać do przykrywania przeciążenia życiowego,
- nie mieszać z alkoholem i lekami uspokajającymi bez wiedzy lekarza,
- nie zwiększać dawki, jeśli nie ma efektu,
- nie ignorować objawów ubocznych.

## Jak oceniać efekt

| Obszar | Co obserwować |
|---|---|
| Stres | napięcie 1–10 |
| Sen | zasypianie, wybudzenia, jakość rano |
| Energia | czy nie zamula |
| Nastrój | czy nie spłaszcza emocji |
| Trening | czy nie obniża napędu |

## Ocena

| Kryterium | Ocena |
|---|---|
| Sens ogólny | C/X |
| Priorytet | ostrożny test |
| Koszt / efekt | osobniczy |
| Ryzyko marketingu | wysokie |
| Ryzyko działań ubocznych | istotne do obserwacji |

## Decyzja

Tylko jako świadomy, ograniczony czasowo test. Jeśli daje apatię, spłaszczenie, problemy żołądkowe albo dziwne objawy — odstawić.
