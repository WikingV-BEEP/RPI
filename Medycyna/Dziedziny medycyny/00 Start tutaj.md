## Zakres działu

Ten folder zastępuje niejasny folder `Rodzaje`. Ma służyć do opisywania dziedzin medycyny, czyli dużych obszarów specjalistycznych.

## Przykładowe dziedziny

| Dziedzina | Zakres |
|---|---|
| dermatologia | skóra, włosy, paznokcie |
| kardiologia | serce i układ krążenia |
| neurologia | układ nerwowy |
| endokrynologia | hormony i gruczoły dokrewne |
| gastroenterologia | przewód pokarmowy |
| pulmonologia | płuca i układ oddechowy |
| nefrologia | nerki |
| psychiatria | zaburzenia psychiczne |
| ortopedia | układ ruchu |
| onkologia | nowotwory |

## Zasada

Notatki o lekarzach i specjalistach idą do `System ochrony zdrowia w Polsce/Lekarze i specjaliści`, a notatki o dziedzinach jako obszarach wiedzy idą tutaj.
